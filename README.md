# Most Cited AI/ML Papers

## Awesome list criteria

A list of **top 100 AI/ML papers** published from 2012 to 2018.

Citation criteria
- Year: **2017** ? citations >= 60
- Year: **2016** ? citations >= 100
- Year: **2015** ? citations >= 200
- Year: **2014** ? citations >= 400
- Year: **2013** ? citations >= 600
- Year: **2012** ? citations >= 800
- Year: **2011 and old** : citations >= 900
